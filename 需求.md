  # 接口文档HTTP / WebSocket API                                                     

  UI 和 API 共用  :13888 。所有 HTTP 路径都有带  /api                         
  前缀和不带前缀两种形式(都能用,新代码建议带  /api )。                        
  权威定义见仓库内  ComfyUI/openapi.yaml (OpenAPI 3.1)。                      
                                                                              
  启动时加  --enable-cors-header '*'  才能让浏览器/其他域的服务跨域调用。     
                                                                              
  ### Workflow 执行(给外部服务最常用的)                                       

   方法   │ 端点                      │ 作用                                
  ────────┼───────────────────────────┼─────────────────────────────────────
    POST  │  /api/prompt              │ 提交 workflow。body:                
          │                           │  {"prompt": <workflow_api_json>, "c 
          │                           │ lient_id": "<uuid>"} ,返回          
          │                           │  {prompt_id, number, node_errors}   
    GET   │  /api/queue               │ 看正在跑 + 等待中 ( queue_running   
          │                           │ /  queue_pending )                  
    POST  │  /api/queue               │ body  {"clear": true}               
          │                           │ 清空队列; {"delete": ["<prompt_id>" 
          │                           │ ]}  删除指定任务                    
    POST  │  /api/interrupt           │ 中断当前正在执行的任务              
    POST  │  /api/free                │ body                                
          │                           │  {"unload_models": true, "free_memo 
          │                           │ ry": true}  卸载模型释放显存        
    GET   │  /api/history             │ 全部历史(默认最多 200 条,可加       
          │                           │  ?max_items=N )                     
    GET   │  /api/history/{prompt_id} │ 单个任务的输出 + 状态               
    POST  │  /api/history             │ body  {"clear": true}  或           
          │                           │  {"delete": [...]}  管理历史        
    GET   │  /api/jobs  /             │ 新版 job 视图(等价于                
          │  /api/jobs/{job_id}       │ history,带更多元数据)               
                                                                              
  典型流程(外部服务调 ComfyUI):                                               
                                                                              
  1. 生成一个  client_id (UUID)。                                             
  2. 连  WS /ws?clientId=<client_id>  监听进度事件。                          
  3.  POST /api/prompt  提交 workflow,拿到  prompt_id 。                      
  4. WebSocket 收到  executed  或  execution_success  事件后, GET             
  /api/history/{prompt_id}  拿输出文件名。                                    
  5.  GET /api/view?filename=...&subfolder=...&type=output  下载图/视频。     
                                                                              

  │ workflow JSON 必须是 "API 格式" —— 在 ComfyUI UI 里点 Settings → "Enable
  │ Dev Mode Options" → 然后 "Save (API Format)" 导出。

  ### WebSocket  /ws                                                          

    ws://<host>:13888/ws?clientId=<your-uuid>                                 

  每条消息是 JSON: {"type": "<event>", "data": {...}} 。常见 event:           
                                                                              
   event                              │ 说明                                 
  ────────────────────────────────────┼─────────────────────────────────────
    status                            │ 队列长度变化                         
    execution_start  /                │ 任务开始/命中缓存                    
    execution_cached                  │                                      
    executing                         │ 当前在跑哪个节点( data.node  为      
                                      │  null  表示结束)                     
    progress                          │ 节点内部进度(采样步数等)             
    executed                          │ 节点完成,带输出引用                  
    execution_success  /              │ 终态                                 
    execution_error  /                │                                      
    execution_interrupted             │                                      
    b_data                            │ 二进制预览图(Sec-WebSocket 子帧)
